# WiFi Edge Node

## Project Overview
This project is part of the WiFi Edge Nodes training program for polyhouse mushroom cultivation monitoring.

The node will collect environmental data such as temperature, humidity, and CO₂ levels and send it to the cloud.

## Hardware Selected
ESP8266 NodeMCU

## Why ESP8266?
- Low cost
- Built-in WiFi
- Suitable for simple IoT sensor nodes

## Repository Structure
- src/ : Source code
- include/ : Header files
- data/ : Data files
